## 📖 Project Overview

This repository contains two React.js applications developed as part of the **Week 04 Minor Project**. The objective of this project is to demonstrate the fundamentals of React.js, including component-based architecture, state management, event handling, React Hooks, and modern JavaScript concepts.

The repository includes:

- **To-Do App** – A task management application for organizing daily activities.
- **Notes App** – A note-taking application for creating, editing, searching, and managing notes.


# 📌 Project 1: To-Do App

### Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Display task status (Completed/Pending)
- Filter tasks (All, Completed, Pending)
- Local Storage integration
- Responsive user interface



# 📝 Project 2: Notes App

### Features
- Create new notes
- Edit existing notes
- Delete notes
- Search notes
- Categorize notes
- Display note timestamps
- Local Storage integration
- Responsive user interface
